Featured - Hackathon iOS App Development

We developed an extension on the Shopee app to achieve a new retail platform that integrates online and offline.

When user take a picture of the product he/she want in the store, ”Image Search” can searches for the corresponding product in Shopee. Then a function allows users to earn gift certificates by completing a game using AR technology.

VIEW PROJECT

Featured - Development iOS (CoreML) App Development

I developed the service that infers shooting spots from image data using CoreML technology. I referred to AWS LocationNet to create the CoreML Model.

Furthermore, the application is created in full scratch from UI design and icon creation.

VIEW PROJECT

Featured - Development iOS (Siri) App Development

I developed an app to support “Journaling” for English learning. Journaling involves the following steps:
1. write your diary in English
2. have a native English teacher correct your writing
3. read the corrected sentences aloud over and over
Pronunciation correction and voice input functions are implemented using Siri technology.

VIEW PROJECT

Featured - Hackathon iOS App + ML Development

We develop a trip sharing platform for travelers to share their original trips together, like Airbnb. We were selected as a finalist for JunctionTokyo 2018.

It can match travelers and individual trip organizers together on the day. And We integrated AI technology to recognize information from images using Cloudinary.

VIEW PROJECT

Featured - Development RPA App with iOS Development

I created a Tutorial to easily integrate RPA business tools using Google Spreadsheet and Google App Script with iOS apps. It shows how the iOS app communicates with the GAS server and how to reference Spreadsheet data.

The demo app can be used as a service to learn terms used in business.
